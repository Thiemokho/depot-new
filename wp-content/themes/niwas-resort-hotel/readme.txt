== Description ==

Niwas Resort Hotel is an attention grabbing, stylish, inviting and modern hotel and resort WordPress theme for luxury hotel, resort, lodge, home stay, motel, guest house, holiday home, inn, dormitory, farmhouse, service apartment and other such hospitality business. Apart from that it can also be used by restaurants, eateries, food joints, cafes and bakeries. It can very well serve bloggers, travel adventures, travel critic and hotel inspector or reviewer. It has a manageable interface of backend which makes designing website a super easy task. It loads fast and gives a good rank to website in search engine results. This resort and hotel WordPress theme is fully responsive and loads on all browsers smoothly. It can be translated into several other languages and you can write content from left to right as well. It facilitates easy customization through theme customizer wherein you can change the colour scheme, background, font, logo, header, footer and menu styles and slider settings. The theme supports gallery where you can show alluring hotel and resort images; testimonial section to get customer feedback and subscriber’s form to connect with customers. The Bootstrap framework strengthens its base. It is a complete theme for all range of hotels and resorts.
